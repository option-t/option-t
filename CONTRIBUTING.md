# Contribution Guide

This guide describes how to contribute to this repository.


## Issues

Thank you for your contribution to help us to improve the quality of this project!
Please file an new issue before you work on a new pull request.


## Pull Requests

- When open a new pull request, a contributor must link the issue which they would resolve.
- A contributor must have a responsibility to an artifact which they submit even if the artifact is generated by AI tools.
- A maintainer might reject your pull request due to a code quality, a project goal, or etc. Sorry.


### AI contributions

- Every commits generated by AI must contains how or who generate it in their commit message as the credit.
